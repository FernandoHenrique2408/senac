4.1- Quais os comandos para configurar usuário e email no git?

git config --global user.name

git config --global user.email

git remote add origin

4.2- Qual a diferença entre git e github?

Enquanto o Git é um software, o GitHub é uma plataforma onde você pode subir seus projetos, criar seu portfólio e compartilhá-lo com o mundo.

4.3- O que faz o comando commit?

O comando git commit captura um instantâneo das mudanças preparadas do projeto no momento. Os instantâneos com commit podem ser considerados versões "seguras" de um projeto, o Git nunca os altera, a menos que você peça a ele.

4.4- O que faz o comando push?

O comando push transfere commits do repositório local a um repositório remoto. É o oposto do comando git fetch , que importa commits para branches locais, enquanto o comando push exporta commits para branches remotos. 4.5- O que faz o comando pull? O comando git pull é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
